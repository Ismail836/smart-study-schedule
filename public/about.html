About the project

Why a grid instead of another to-do app

Most planning apps show a list. A list tells you what's left; it doesn't tell you when. Smart Study Schedule was built as a semester project to test a simpler idea: lay the whole week out as a grid, hour by hour, and let color do the prioritizing.

Purpose

A course project, built like a product

This site was developed for a Web Technologies semester project. The brief started as a static, multi-page front-end built with HTML, CSS, JavaScript, and Bootstrap. The finished project keeps that front-end structure, but the schedule builder now uses an Express API and SQLite so the plan persists across visits. Rather than a set of disconnected demo pages, it's built as one coherent tool: a home page that explains the idea, a working builder that delivers on it, and supporting pages that round out the experience.

The interactive weekly grid is the centerpiece. It uses the browser to present the grid, but the saved plan lives in the SQLite database on the server.
